🚀 Cómo levantar el laboratorio

Un comando por stack. Copia, pega, y espera que Docker construya las imágenes. Primera vez: 3–8 minutos. Desde caché: 20–40 segundos.

🐘 Stack PHP completo

Portal + nginx hub + 12 casos PHP + PostgreSQL (01–02) + Prometheus + Grafana

Levantar

docker compose -f compose.root.yml up -d --build

Verificar

docker compose -f compose.root.yml ps

Bajar

docker compose -f compose.root.yml down

Acceso: Portal → localhost:8080 · Casos → localhost:8100/01/ … localhost:8100/12/

🐍 Stack Python completo

Un contenedor · dispatcher interno · 12 casos · stdlib pura · sin dependencias externas

Levantar

docker compose -f compose.python.yml up -d --build

Verificar

docker compose -f compose.python.yml ps

Bajar

docker compose -f compose.python.yml down

Acceso: Todos los casos → localhost:8200/01/ … localhost:8200/12/
